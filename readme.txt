== Description ==

Contact Forms Entries Plugin automatically saves form submissions from Contact Form 7, CRM Forms and many other popular contact form plugins to wordpress database. Learn more at [crmperks.com](https://www.crmperks.com/plugins/contact-form-plugins/crm-forms/?utm_source=wordpress&utm_medium=directory&utm_campaign=readme)

== Supported Contact Forms ==

* [Contact Form 7](https://wordpress.org/plugins/contact-form-7/)

== Key Features ==

* You can view all entries in default wp table form.
* Search by all or specific field.
* Filter entries by Date.
* Mark entries read/un-read.
* Save favorite entries.
* Pint entries.
* Select table columns from "Screen Options".
* File field of an entry supports multiple files.
* Add, edit notes to any entry.
* Send note to an email address. 
* Display entries on any wordpress page in sortable table form.
* Export entries in csv format.
* Display entries stats on dashboard.
